# How to Download Wallpapers
_Click the **"Click Here** to Download Wallpapers" button from each titles / pack below._

---

Alternatively, _this is the manual method to download the wallpapers from a specific folder._
1. **Go to the folder** you wanted to download.
2. **Copy** the URL Path / the Link.
3. Go to https://downgit.github.io/
4. **Paste** the link.
5. Click **"Download"**.
